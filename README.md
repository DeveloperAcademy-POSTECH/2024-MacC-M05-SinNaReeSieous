## 1. 앱 소개
> ✨집집사, 어떤 앱인가요?

집을 보러 간 상황에서 체크리스트로 필수사항을 빠르게 기록하고, 룸플랜으로 집의 구조를 기록하여 하나의 앱 안에서 집 별로 정보를 관리할 수 있는 앱입니다. 

> ✨타겟
- 직장 때문에 이사하는 사회 초년생
- 처음 집을 구해보는 사람
- 집 구할 때 놓치는 정보없이 꼼꼼하게 보고 싶은 사람

> ✨어떤 상황에서 쓸 수 있나요?
- 집 보러가서 정보를 기록할 때
- 집 내부 사진을 못 찍을 때
- 집 보는 것이 처음이라 무엇을 체크해야할지 모를 때
- 집에 돌아와서 각 집에 대한 정보를 확인할 때

## 2. 주요 기능
### 📋 체크리스트
- 집집사는 사용자 맞춤형 체크리스트를 제공해요.
- 방충, 청결, 치안, 환기, 방음, 채광 중에서 중요한 카테고리를 선택하면 추가 질문과 설명을 챙겨드려요.

### 📱 집 구조 기록
- 아이폰 내장 기술인 RoomPlan을 사용해 집의 구조를 기록해요.
- 집 구조를 3D 모델 형태로 보여주기 때문에 세입자가 있어 사진을 찍기 어려운 상황이거나, 집의 구조를 객관적으로 파악하고 싶을 때 활용할 수 있어요.
> ❗️ RoomPlan은 라이다 센서가 있는 아이폰 12 프로 모델부터 지원해요

### 💳 집 요약 카드
- 사회초년생 특성 상, 혼자서 집을 결정하는 경우보다는 부모님이나 친구들과 상의하는 경우가 많아요.
- 하나의 집을 보고나면 하나의 카드로 정보가 정리되어 나와요.
- 이 카드 자체를 부모님이나 친구들에게 공유하여 의견을 받을 수 있도록 했어요.
